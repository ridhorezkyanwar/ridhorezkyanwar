## 🚀 Proyek yang saya banggakan

### [Sentinel Log Analyzer](https://github.com/ridhorezkyanwar/sentinel-log-analyzer)

**AI-powered SOC Copilot** — ini proyek favorit saya.

**Masalah:** Saya capek manual baca log server buat nyari security threat. Ribet banget!

**Solusi:** Bikin tool yang otomatis analisis log pakai LLM (Llama 3.3 via Groq API). Bisa extract IoC (Indicators of Compromise) secara real-time.

**Tech:** Python, Groq API, Vercel AI SDK, TypeScript

**Live:** [sentinel-log-analyzer-phi.vercel.app](https://sentinel-log-analyzer-phi.vercel.app)

**Yang saya pelajari:** Prompt engineering itu seni. Butuh 15+ iterasi sebelum dapat prompt yang bagus. Dan streaming responses itu wajib buat UX yang baik.
